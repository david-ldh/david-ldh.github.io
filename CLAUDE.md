# Pagina Web Personal — david-ldh.github.io

## Objetivo
Construir una página web personal publicada en GitHub Pages bajo el dominio `https://david-ldh.github.io`.

## Repositorio
- GitHub: `david-ldh/david-ldh.github.io`
- Publicación automática desde la rama `main`

## Estructura del proyecto
```
Pagina_web_personal/
├── index.html        # Página principal
├── css/
│   └── styles.css    # Estilos globales
└── CLAUDE.md         # Este archivo
```

## Convenciones
- HTML semántico con secciones: `#hero`, `#sobre-mi`, `#proyectos`, `#contacto`
- CSS con variables custom (`:root`) para colores y espaciados
- Paleta oscura estilo GitHub (`--color-bg: #0d1117`)
- Sin frameworks ni build tools — HTML/CSS/JS plano para mantener la simplicidad de GitHub Pages
- Español como idioma principal del contenido
